3️⃣ Ejercicio: Sensor de Temperatura 🌡

Crea una clase Sensor con la propiedad privada #temperatura.

Implementa un método para leer la temperatura.

Implementa un método para cambiar la temperatura (solo si está entre -50 y 100 grados).

Crea una instancia de Sensor, intenta cambiar la temperatura con valores correctos e incorrectos y verifica el resultado.
